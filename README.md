# TaskFlow Dashboard

A lightweight task and project management dashboard for teams. Kanban board, productivity stats, and priority filters.

## Features

- 📝 **Kanban board** — drag tasks between To Do, In Progress, and Done
- 📊 **Team stats** — tasks by status, priority, and assignee
- 🏷️ **Priority labels** — low, medium, high
- 👤 **Assignees** — track who is working on what
- 💾 **Local persistence** — data saved to browser localStorage (demo mode)
- 📱 **Responsive** — works on desktop and mobile

## Tech stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000
